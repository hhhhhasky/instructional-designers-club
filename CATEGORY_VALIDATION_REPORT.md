## 4. 数据映射关系

### 前后端字段对应

```
数据库层（course_categories表）
├─ id: UUID
├─ name: 分类名称 ✓ → API返回
├─ description: 分类描述
├─ sort_order: 排序顺序 ✓ → 控制显示顺序
├─ created_at: 创建时间
└─ updated_at: 更新时间

API层（getCourseCategories）
├─ 查询: SELECT name FROM course_categories ORDER BY sort_order
├─ 处理: 提取分类名称
└─ 返回: ["全部", ...分类名称]

前端层（CoursesPage）
├─ 状态: categories: string[]
├─ 显示: 分类Tab按钮
└─ 交互: 点击筛选课程
```
