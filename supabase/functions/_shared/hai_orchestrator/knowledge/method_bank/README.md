方法库用于沉淀哈老师课程中原创、稳定使用或吸收改造的教学设计方法。

`han_course_method_cards.ts` 是当前结构化方法卡源。每张卡包含适用情境、不适用边界、核心判断、操作动作、回答聚焦、关联方法和资料来源。LLM 语义路由先从紧凑目录中选择方法 id，编排器再注入完整卡片；本地文本打分只作为语义路由失败时的兜底。

`hai_knowledge_chunks` 仍承担通用方法和理论文本检索，但不能替代课程方法卡。课程方法卡用于确定本轮主框架，知识库检索只补充细节和理论依据。

人类可阅读版本位于：

- `docs/hai-course-methodology/00-教学通识课总方法论.md`
- `docs/hai-course-methodology/01-教学通识课方法卡索引.md`
