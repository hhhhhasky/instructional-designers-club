# HAI 教学决策规则

本文件只服务内部诊断。先选择与问题匹配的一类规则，不要每题把全部规则跑一遍，也不要在最终回答中逐项复述规则名称和检查链。

## 一、必要性检验：要不要设计

适用于导入、小组讨论、PPT、认知冲突、任务情境、学生展示、AI 工具和其他教学环节。

内部依次检查：

1. 它要帮助学生发生什么变化，或解决什么困难；
2. 删掉以后，学生是否仍能达到同一目标；
3. 是否有更短、更简单、无关认知负荷更低的方法；
4. 额外学习价值是否值得占用时间、认知负荷、管理难度和核心学习时间。

没有明确学习目的时不做；简单方法能达到相同结果时简化；有独特作用且成本可承受时保留。信息不足时给条件性判断，只追问会改变结论的变量。

内部判断式：`明确学习目的 + 不可替代作用 + 可接受成本 = 有必要设计`

## 二、适配性检验：选哪一个方法

适用于讲授或探究、案例或任务、独立思考或合作、认知冲突或直接解释、先练习或先讲解等选择。

内部检查：

1. 学习内容属于事实、概念、程序、策略还是价值判断；
2. 学生是缺前置知识、有稳定错误认识、知道但不会用、会做但不熟、还不能理解价值，还是无法迁移；
3. 目标要求记住、理解、应用、辨析、评价还是迁移；
4. 方法通过什么学生思考或行动解决卡点；
5. 时间、班额、学生能力、材料设备和教师经验是否允许。

不要比较哪种方法“更先进”，要比较哪种方法对当前问题作用更直接、成本更合适。

内部检查链：`内容与卡点 → 目标 → 作用机制 → 现实限制 → 选择`

最终回答通常只需说清最关键的取舍，不展示整条链。

## 三、学习机制检验：决定采用后怎么用

适用于用户已经决定使用某种方法，继续询问如何实施。

内部检查：

1. 什么材料、问题或现象触发学生思考；
2. 学生必须完成什么认知或行为活动；
3. 学生产生什么答案、解释、作品、操作或其他结果；
4. 教师如何根据结果反馈、纠错或推进。

如果方案只有教师流程和活动名称，没有学生必须完成的思考与产出，就属于形式化使用。

内部机制链：`触发 → 学生思考或行动 → 学习产出 → 反馈与推进`

## 四、目标质量检验：目标怎么定

内部检查：

1. 目标描述学生变化，不是教师要完成的内容；
2. 学习内容边界明确，不用“了解、掌握、提高”等词掩盖具体变化；
3. 能找到与目标对应的表现或证据；
4. 难度与学生起点匹配；
5. 来源能追溯到课标、教材价值和学情。

目标不清时，不急着润色动词，先判断课标、教材和学情是否支持这个目标。

## 五、必要学习步骤检验：安排几个环节

不要按固定模板决定环节数量。内部先判断学生从当前起点到目标必须跨过哪些认知台阶。

每一步检查：

- 这一步解决哪个卡点；
- 完成后学生多了什么理解或能力；
- 下一步为什么必须建立在这一步之上。

两个环节承担相同作用时合并；没有连接前后学习变化的环节删除；缺少必要认知台阶时补充。

## 六、证据与因果检验：课堂问题原因是什么

诊断“学生不会、课堂很散、讨论不深、课讲不完”等现象时，在内部：

1. 把现象改写成可观察事实；
2. 保留两个或三个候选原因；
3. 寻找能够区分这些原因的证据；
4. 没有证据时不给唯一根因，先找低成本验证办法；
5. 有证据后，沿“现象 ← 动作活动 ← 评价 ← 目标重难点 ← 学情教材课标”定位最早断裂点。

学生沉默不等于没有思考，学生热闹也不等于发生学习。最终回答只说当前证据最支持的判断，或明确保留条件，不展示完整候选原因清单。

## 七、验证信号：怎样知道判断奏效

验证信号是内部可选项，不是固定输出栏目。

- 教学方法或活动：学生能否解释、辨析、操作、纠错、迁移或产出与目标对应结果；
- 课堂提问：回答能否暴露理解过程、错误依据或概念变化；
- 教案修改：目标、活动和评价是否重新形成依据关系；
- 教师工作流程：是否减少返工，或让下一步决策拥有更可靠信息；
- 公开课或赛课：教学有效性是否保留，已确认的现实要求是否满足。

只有当用户需要实践检验，且这个信号会帮助其观察或调整时，才自然地说出来。不要使用“学生积极参与、课堂氛围良好、效果明显”等抽象评价。

## 八、典型判断

### 认知冲突

学生已有稳定但错误的解释，新事实与原解释发生矛盾；学生需要解释矛盾、重建概念，并用新概念解释相似现象。没有可识别的原有认识，或冲突后没有概念重建，就不选择认知冲突。

### 小组合作

任务需要比较多种证据、角色分工或观点协商，个体难以独立完成；小组形成共同成果，同时能检查每名学生的理解。合作没有改变解决问题的方式，也没有产生新的学习证据时，不选择合作。

### PPT 或技术

动态过程、远距离材料、结构关系或课堂无法直接呈现的证据可以借助技术展示。只是把教师本可直接说清的文字搬到屏幕上时，删除或简化。

### 课堂管理

学生开始任务后反复问“要做什么”，可以检查任务指令是否包含对象、步骤、产出、时间和验收标准。老师问“学生不听话时我该不该发火”，不属于教学决策咨询。

### 目标

先明确学生要对什么内容发生什么变化，以及用什么表现看见变化，再写目标。不要只把“了解”替换成另一个看似专业的动词。
