# 首页统计数据验证指南

## 🎯 快速验证

### 验证目标

确认首页统计数据已从基于时间的递增逻辑改为从数据库实时获取。

## 📝 验证步骤

### 步骤1：验证数据库统计

**在 Supabase SQL Editor 中执行：**

```sql
-- 查询当前的统计数据
SELECT 
  (SELECT COUNT(*) FROM course_categories) as categories_count,
  (SELECT COUNT(*) FROM courses WHERE status = 'published') as published_courses_count,
  (SELECT SUM(duration) FROM courses WHERE status = 'published') as total_duration;
```

**预期结果：**
```
categories_count: 9
published_courses_count: 36
total_duration: 3299
```

### 步骤3：验证动态更新

**测试A：添加新课程**

**操作：**
1. 在数据库中添加测试课程

```sql
-- 添加测试课程
INSERT INTO courses (
  title,
  description,
  category,
  level,
  duration,
  status
) VALUES (
  '【验证测试】统计数据更新测试课程',
  '这是一门用于验证统计数据更新的测试课程',
  '建构主义',
  '中级',
  90,
  'published'
);

-- 查询更新后的统计数据
SELECT 
  (SELECT COUNT(*) FROM courses WHERE status = 'published') as published_courses_count,
  (SELECT SUM(duration) FROM courses WHERE status = 'published') as total_duration;
```

**预期结果：**
- 课程节数：36 → 37（+1）
- 累计时长：3299 → 3389（+90）

2. 刷新首页（Ctrl+R 或 Cmd+R）
3. 观察统计数据是否更新

**验证结果：**
- [ ] 课程节数增加1
- [ ] 累计时长增加90
- [ ] 学习营数量不变

**测试B：添加新分类**

### 步骤4：清理测试数据

**操作：**

```sql

-- 删除测试分类

WHERE name LIKE '%验证测试%';

-- 验证清理结果

  (SELECT COUNT(*) FROM course_categories) as categories_count,
  (SELECT COUNT(*) FROM courses WHERE status = 'published') as published_courses_count,

```

**预期结果：**
- 恢复到原始数据：9个分类，36门课程，3299分钟
