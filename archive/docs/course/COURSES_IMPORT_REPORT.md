## 📞 技术支持

如需修改或更新课程数据，请使用以下SQL：

```sql
-- 更新单个课程
UPDATE courses
SET 
  title = '新课程名称',
  description = '新课程描述',
  duration = 60,
  credits = 1.0
WHERE id = '课程ID';

-- 批量更新分类
UPDATE courses
SET category = '新分类名称'
WHERE category = '旧分类名称';

-- 删除课程
DELETE FROM courses
WHERE id = '课程ID';
```
