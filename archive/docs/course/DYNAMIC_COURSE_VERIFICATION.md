## 🔄 完整数据流程验证

### 流程1：添加新课程 → 课程中心显示

```
1. 在数据库中添加新课程
   ↓
2. 设置 status = 'published'
   ↓
3. 用户访问 /courses
   ↓
4. CoursesPage 组件加载
   ↓
5. useEffect 触发，调用 getCourses()
   ↓
6. API 查询：SELECT * FROM courses WHERE status = 'published'
   ↓
7. 返回37门课程（包括新课程）
   ↓
8. setState({ courses: [...] })
   ↓
9. 渲染37个课程卡片
   ↓
10. 新课程卡片显示在页面上 ✅
```

**验证结果：** ✅ 流程完整，功能正常

### 流程2：点击课程卡片 → 课程详情页

```
1. 用户点击新课程卡片
   ↓
2. handleCourseClick(courseId) 触发
   ↓
3. navigate(`/courses/${courseId}`)
   ↓
4. 路由跳转到 /courses/46294536-5098-4ad8-8da9-8dc1a163238e
   ↓
5. CourseDetailPage 组件加载
   ↓
6. useParams 获取课程ID
   ↓
7. useEffect 触发，调用 getCourseById(id)
   ↓
8. API 查询：SELECT * FROM courses WHERE id = '...'
   ↓
9. 返回课程详情
   ↓
10. setState({ course: {...} })
   ↓
11. 调用 incrementCourseViewCount(id)
   ↓
12. RPC 更新：view_count = view_count + 1
   ↓
13. 渲染课程详情页 ✅
```

**验证结果：** ✅ 流程完整，功能正常
