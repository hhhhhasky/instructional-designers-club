### 2. 微信平台验证流程

**步骤：**

1. **登录微信公众平台或开放平台**
   - 访问：https://mp.weixin.qq.com/ 或 https://open.weixin.qq.com/

2. **进入域名验证页面**
   - 找到"设置" → "公众号设置" → "功能设置"
   - 或者"开发" → "接口权限" → "网页授权域名"

3. **添加域名**
   - 输入您的域名（不包含 http:// 或 https://）
   - 例如：`example.com`

4. **下载校验文件（已完成）**
   - 微信会提供一个校验文件
   - 我们已经创建了这个文件：`80c26dc8a6320a9717670f56027ee3c6.txt`

5. **上传校验文件（已完成）**
   - 文件已放置在网站根目录
   - 可通过 `域名/80c26dc8a6320a9717670f56027ee3c6.txt` 访问

6. **点击验证**
   - 在微信平台点击"验证"按钮
   - 微信会访问您的域名验证文件
   - 验证通过后，域名即可使用

7. **保持文件**
   - ⚠️ 验证通过后，建议保留此文件
   - 微信可能会定期重新验证

### 3. 常见问题

**问题1：验证失败，提示"文件不存在"**

**原因：**
- 应用未部署到生产环境
- 文件路径不正确
- 服务器配置问题

**解决方案：**
1. 确认应用已部署
2. 通过浏览器访问文件URL
3. 检查服务器配置（Nginx、Apache等）

**问题2：验证失败，提示"文件内容不正确"**

**原因：**
- 文件内容被修改
- 文件编码问题
- 多余的空格或换行

**解决方案：**
1. 检查文件内容是否完全一致
2. 确保文件编码为 UTF-8
3. 确保没有多余的空格或换行

**问题3：验证失败，提示"无法访问"**

**原因：**
- 域名未解析
- 防火墙拦截
- SSL证书问题

**解决方案：**
1. 检查域名DNS解析
2. 检查防火墙规则
3. 检查SSL证书是否有效

### Vite 静态资源处理

**Vite 配置：**
- `public/` 目录中的文件会被直接复制到输出目录的根路径
- 不会经过任何处理或打包
- 可以直接通过根路径访问

**构建过程：**
```bash
npm run build
# 输出：
# dist/
#   ├── index.html
#   ├── assets/
#   └── 80c26dc8a6320a9717670f56027ee3c6.txt  ← 自动复制
```

### 服务器配置

**Nginx 配置示例：**
```nginx
server {
    listen 80;
    server_name example.com;
    root /var/www/html;

    # 静态文件
    location / {
        try_files $uri $uri/ /index.html;
    }

    # 确保 .txt 文件可以访问
    location ~* \.txt$ {
        add_header Content-Type text/plain;
    }
}
```

**Apache 配置示例：**
```apache
<VirtualHost *:80>
    ServerName example.com
    DocumentRoot /var/www/html

    <Directory /var/www/html>
        Options Indexes FollowSymLinks
        AllowOverride All
        Require all granted
    </Directory>

    # 确保 .txt 文件可以访问
    <FilesMatch "\.txt$">
        Header set Content-Type "text/plain"
    </FilesMatch>
</VirtualHost>
```
