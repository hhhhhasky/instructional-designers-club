## 技术细节（可选阅读）

### 什么是 Edge Function？

Edge Function 是运行在服务器端的函数，不受浏览器 CORS 限制。我们创建了一个代理函数，前端调用这个函数，函数再去访问数据库，完美绕过了 CORS 问题。

### 为什么这样可以解决问题？

- **CORS 限制**只存在于浏览器访问跨域资源时
- **Edge Function** 运行在服务器端，不受 CORS 限制
- 通过代理，我们把跨域请求变成了同域请求

### 有没有性能影响？

- 延迟增加约 50-100 毫秒
- 完全可以忽略（访问统计是异步操作）
- 用户完全无感知

### 有没有后备方案？

有！如果 Edge Function 出现问题，系统会自动尝试直接调用数据库（原方案），确保访问统计始终可用。
