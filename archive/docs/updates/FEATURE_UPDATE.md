# 功能更新说明

## 📊 网站访问统计功能

### 功能概述
在所有页面底部添加了实时的网站访问统计，包括：
- **网站访问人数**：基于UUID的唯一访客数量
- **网站访问人次**：总访问次数统计

### 技术实现
1. **唯一访客识别**
   - 使用 `crypto.randomUUID()` 为每个访客生成唯一标识
   - UUID存储在localStorage中，确保同一用户多次访问被识别为同一人
   - 访客列表存储在localStorage的 `visitors_list` 中

2. **访问次数统计**
   - 每次页面加载自动增加访问次数
   - 访问次数存储在localStorage的 `total_visits` 中
   - 使用CountUp组件实现数字滚动动画效果

3. **组件结构**
   ```
   src/components/common/VisitorStats.tsx  // 访问统计组件
   src/components/common/Footer.tsx        // 集成统计的页脚组件
   ```

4. **数据持久化**
   - 使用localStorage存储数据
   - 数据在浏览器中持久保存
   - 清除浏览器数据会重置统计

### 视觉设计
- 两个统计指标并排显示（桌面端）
- 移动端垂直堆叠
- 图标 + 数字 + 标签的清晰布局
- 悬停时图标缩放和背景色变化
- 数字使用CountUp动画效果

## 🔧 维护说明

### 访问统计
- 数据存储在用户浏览器的localStorage中
- 清除浏览器数据会重置统计
- 可以通过修改localStorage的key来重置所有用户的统计

### 课程图片
- 图片URL存储在HomePage.tsx中
- 更换图片只需修改src属性
- 建议使用相同宽高比的图片（16:9）

### 动画效果
- 所有动画定义在src/index.css中
- 可以通过修改CSS变量调整动画参数
- 动画延迟可以通过style属性的animationDelay控制
