**状态**：已完成  
**提交**：2afe32e

#### 4.1 自动更新规则
1. **学习营数量**：每月自动增加1个
2. **课程节数**：每周自动增加1节
3. **累计时长**：自动计算为课程节数 × 60分钟

#### 4.2 技术实现
- **起始时间**：2025年1月1日
- **起始数据**：学习营6个，课程31节
- **计算方法**：
  - 月份差 = 当前月份 - 起始月份
  - 周数差 = (当前时间 - 起始时间) / 7天
  - 累计时长 = 课程节数 × 60
