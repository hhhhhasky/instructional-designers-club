## 🎨 技术实现

### 统一的图片展示组件结构

```tsx
<Card className="border-border shadow-[var(--shadow-elegant)] mb-12 overflow-hidden hover-lift hover-glow animate-scale-in">
  <CardContent className="p-0">
    <img 
      src="图片URL" 
      alt="图片描述" 
      className="w-full h-auto transition-transform duration-500 hover:scale-105"
    />
  </CardContent>
</Card>
```

### 样式说明

**Card容器样式**：
- `border-border`：使用主题边框颜色
- `shadow-[var(--shadow-elegant)]`：优雅的阴影效果
- `mb-12`：底部间距12单位
- `overflow-hidden`：隐藏溢出内容（配合图片缩放）
- `hover-lift`：鼠标悬停时卡片上浮效果
- `hover-glow`：鼠标悬停时发光效果
- `animate-scale-in`：入场缩放动画

**图片样式**：
- `w-full`：宽度100%
- `h-auto`：高度自适应
- `transition-transform duration-500`：平滑的变换过渡（500ms）
- `hover:scale-105`：鼠标悬停时放大5%
