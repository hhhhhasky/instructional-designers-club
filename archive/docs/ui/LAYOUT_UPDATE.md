## 🎯 调整逻辑

### 1. 认知流程优化
按照用户的认知和决策流程重新排列内容：

**了解阶段**
- 俱乐部介绍：让用户快速了解俱乐部是什么
- 创始人介绍：建立信任，展示专业性

**兴趣阶段**
- 课程介绍：展示核心价值，激发兴趣
- 课程试看：提供具体内容，加深了解

**信任阶段**
- 俱乐部数据：用数据证明价值
- 会员简介：展示优秀学员
- 会员评价：社会证明，增强信任

**决策阶段**
- 常见问题：解答最后的疑虑
- 立即加入：明确的行动号召

### 2. 内容层次优化

**顶部（吸引注意）**
- Hero Section：欢迎信息
- 俱乐部介绍：核心定位
- 创始人介绍：建立权威

**中部（展示价值）**
- 课程介绍：核心产品
- 课程试看：具体内容
- 俱乐部数据：量化价值

**底部（促进转化）**
- 会员简介：成功案例
- 会员评价：用户口碑
- 常见问题：消除顾虑
- 立即加入：行动号召

## ✨ 新增"立即加入"Section

### 设计特点

**视觉设计**
- 渐变背景：`from-primary/10 via-background to-primary/5`
- 突出显示，与其他section区分
- 营造温暖、邀请的氛围

**内容结构**
1. **标题区域**
   - 主标题：开启你的教学设计之旅
   - 副标题：价值主张说明

2. **三大价值点**
   - 系统化学习：每周一课、每月一营
   - 专业社群：与优秀教育者交流
   - 持续更新：紧跟教育前沿

3. **行动按钮**
   - 大号按钮：`size="lg"`
   - 清晰文案：立即加入俱乐部
   - 跳转说明：点击按钮填写报名表单

**交互效果**
- 卡片悬停上浮和发光
- 按钮悬停阴影增强
- 渐进式动画进入
- 图标渐变背景

### 代码实现

```tsx
<section id="join" className="py-16 xl:py-24 px-4 bg-gradient-to-br from-primary/10 via-background to-primary/5">
  <div className="max-w-4xl mx-auto text-center">
    {/* 标题区域 */}
    <div className="mb-8 animate-fade-in">
      <h2>开启你的教学设计之旅</h2>
      <p>加入教学设计师俱乐部...</p>
    </div>

    {/* 三大价值点 */}
    <div className="grid grid-cols-1 xl:grid-cols-3 gap-6 mb-12">
      {/* 系统化学习 */}
      {/* 专业社群 */}
      {/* 持续更新 */}
    </div>

    {/* CTA按钮 */}
    <Button onClick={() => window.open('报名链接', '_blank')}>
      立即加入俱乐部
    </Button>
  </div>
</section>
```

## 🎨 视觉优化

### 背景色交替
为了提升视觉层次和阅读体验，各section采用交替背景：

- 俱乐部介绍：`bg-secondary/30`（浅色）
- 创始人介绍：`bg-background`（默认）
- 课程介绍：`bg-secondary/30`（浅色）
- 课程试看：`bg-background`（默认）
- 俱乐部数据：`bg-secondary/30`（浅色）
- 会员简介：`bg-background`（默认）
- 会员评价：`bg-secondary/30`（浅色）
- 常见问题：`bg-background`（默认）
- 立即加入：`bg-gradient-to-br`（渐变，突出）

### 间距统一
- 常规section：`py-12 xl:py-16`
- 立即加入section：`py-16 xl:py-24`（加大间距，突出重要性）
