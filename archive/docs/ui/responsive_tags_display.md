# 课程分类卡片标签响应式显示

## 功能概述

课程分类卡片的标签（适用场景、内容类型、适用人群）根据设备屏幕尺寸自动调整显示：
- **桌面端（≥768px）**：显示所有标签
- **移动端（<768px）**：隐藏标签

## 实现方式

### Tailwind CSS 响应式类名

```tsx
<div className="hidden md:flex flex-wrap gap-1.5 mb-2">
  {/* 标签内容 */}
</div>
```

**类名说明**：
- `hidden`：默认隐藏（移动端）
- `md:flex`：中等屏幕及以上显示为 flex 布局（桌面端）

### 断点定义

| 设备类型 | 屏幕宽度 | 标签显示 |
|---------|---------|---------|
| 手机 | 0px - 767px | ❌ 隐藏 |
| 平板/桌面 | 768px 及以上 | ✅ 显示 |

## 用户体验对比

### 移动端（<768px）

**卡片布局**：
```
┌─────────────────────────────┐
│ [图标] 分类名称              │
│        共 X 节课程      [→] │
└─────────────────────────────┘
```

**优势**：
- ✅ 简洁布局，提升可读性
- ✅ 减少内容拥挤，优化触摸体验
- ✅ 卡片高度减少约 40-50%
- ✅ 加快渲染速度

### 桌面端（≥768px）

**卡片布局**：
```
┌─────────────────────────────┐
│ [图标] 分类名称              │
│        [标签1] [标签2] [标签3]│
│        [标签4] [标签5]       │
│        共 X 节课程      [→] │
└─────────────────────────────┘
```

**优势**：
- ✅ 显示完整标签信息
- ✅ 提供更多上下文信息
- ✅ 充分利用屏幕空间
- ✅ 保持视觉丰富度

## 技术优势

### 1. 纯 CSS 实现
- 无需 JavaScript 检测屏幕尺寸
- 无需 React 状态管理
- 无需监听 resize 事件
- 性能最优

### 2. 自动响应
- 浏览器自动根据屏幕宽度应用样式
- 窗口大小改变时自动调整
- 无需手动触发

### 3. 代码简洁
- 只需一个响应式类名
- 易于理解和维护
- 符合 Tailwind CSS 最佳实践

## 未来扩展

### 1. 更多断点

可以添加更多断点，实现更精细的控制：

```tsx
{/* 小屏隐藏，中屏显示部分，大屏显示全部 */}
<div className="hidden md:flex lg:flex-wrap">
  {/* 标签内容 */}
</div>
```

### 2. 标签数量限制

可以在不同屏幕尺寸显示不同数量的标签：

```tsx
{/* 中屏显示前 3 个，大屏显示全部 */}
<div className="hidden md:flex">
  {tags.slice(0, 3).map(...)}
  <span className="hidden lg:inline">
    {tags.slice(3).map(...)}
  </span>
</div>
```

### 3. 标签折叠

可以在桌面端添加"展开/收起"功能：

```tsx
{/* 默认显示前 5 个，点击展开显示全部 */}
<div className="hidden md:flex">
  {showAll ? tags.map(...) : tags.slice(0, 5).map(...)}
  <button onClick={() => setShowAll(!showAll)}>
    {showAll ? '收起' : '展开'}
  </button>
</div>
```
