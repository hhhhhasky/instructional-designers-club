**内边距**

  - 桌面端：`px-8` (2rem) → `px-16` (4rem)
- 垂直内边距：
  - 移动端：`py-6` (1.5rem) → `py-8` (2rem)
  - 桌面端：`py-6` (1.5rem) → `py-10` (2.5rem)

**视觉效果**
- 按钮整体面积增大约 **40-50%**
- 在页面中更加突出和显眼
- 更容易点击，提升移动端用户体验

### 2. 新增脉冲式动画效果

**动画特性**
- 动画名称：`animate-pulse-cta`
- 动画周期：2.5秒
- 缓动函数：`ease-in-out`
- 循环方式：无限循环（infinite）

**动画效果**
- **0% 和 100%（初始/结束状态）**
  - 正常尺寸（scale: 1）
  - 基础发光阴影
  - 阴影范围：0px
  - 发光强度：中等

- **50%（中间状态）**
  - 轻微放大（scale: 1.02）
  - 阴影向外扩散至15px
  - 阴影逐渐淡出（透明度从0.7到0）
  - 发光效果增强（透明度从0.5到0.8）

**视觉体验**
- 柔和的脉冲光晕效果
- 按钮仿佛在"呼吸"
- 吸引注意力但不刺眼
- 闪烁频率适中，不会造成视觉疲劳

## 🎨 CSS实现细节

### 动画类定义

```css
/* CTA按钮脉冲颜色动画 */
.animate-pulse-cta {
  animation: pulseCTA 2.5s ease-in-out infinite;
}
```

### 关键帧动画

```css
@keyframes pulseCTA {
  0%, 100% {
    box-shadow: 0 0 0 0 hsl(var(--primary) / 0.7), 0 0 30px hsl(var(--primary-glow) / 0.5);
    transform: scale(1);
  }
  50% {
    box-shadow: 0 0 0 15px hsl(var(--primary) / 0), 0 0 50px hsl(var(--primary-glow) / 0.8);
    transform: scale(1.02);
  }
}
```

### 技术要点

1. **双层阴影系统**
   - 第一层：扩散阴影（从0px扩散到15px）
   - 第二层：发光阴影（从30px增强到50px）

2. **使用CSS变量**
   - `hsl(var(--primary))` - 主题色
   - `hsl(var(--primary-glow))` - 发光色
   - 确保动画适配明暗主题

3. **透明度渐变**
   - 扩散阴影：0.7 → 0（淡出效果）
   - 发光阴影：0.5 → 0.8（增强效果）

4. **微妙的缩放**
   - scale(1) → scale(1.02)
   - 仅放大2%，避免过于夸张
   - 配合阴影变化，增强立体感

## 🎨 设计原则

### 1. 渐进增强
- 基础功能不依赖动画
- 动画作为视觉增强
- 即使动画不支持，按钮仍可正常使用

### 2. 性能优先
- 使用CSS动画而非JavaScript
- 利用GPU加速
- 避免重排和重绘

### 3. 用户体验
- 动画频率适中（2.5秒）
- 不会造成视觉疲劳
- 不影响页面阅读

### 4. 品牌一致性
- 使用主题色系
- 与现有动画风格统一
- 保持设计系统的完整性
