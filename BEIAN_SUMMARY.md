## ⚠️ 重要提醒

### 1. 必须替换备案号

**当前状态：** 使用占位符 `XXXXXXXX` 和 `XXXXXXXXXXXXX`

**必须操作：** 替换为您的实际备案号

**否则：** 不符合备案要求，可能导致网站被关闭

### 2. 备案号格式

**ICP备案号格式：**
- 正确：`京ICP备12345678号`
- 错误：`京ICP备12345678`（缺少"号"）
- 错误：`ICP备12345678号`（缺少省份）

**公安备案号格式：**
- 正确：`京公网安备 11010802012345号`
- 错误：`京公网安备11010802012345号`（缺少空格）
- 错误：`公网安备 11010802012345号`（缺少省份）

### 3. 链接安全

**必须包含的属性：**
```tsx
target="_blank"           // 在新标签页打开
rel="noopener noreferrer" // 安全性考虑
```

### 4. 显示位置

**正确位置：** 页面底部 Footer

**不要放在：** 页面顶部、侧边栏、浮动窗口
