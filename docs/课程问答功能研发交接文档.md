## 1. 已确认决策

- 入口：课程详情页内，不新增论坛页。
- 展示：同页 Accordion 折叠问答，不做独立帖子页。
- 内容：纯文本问题 + 单层纯文本回复。
- 隐私：发问题/回复时提供匿名开关；匿名内容前台只显示“匿名”。
- 权限：只有登录且有课程访问权限的用户可查看、提问、回复。
- 标签：v1 只做“课程体系”预设标签，不开放用户自建标签。
- 审核：即时发布，管理员后置隐藏/删除。

## 4. 数据与权限说明

- `public.can_access_course(course_id)`：按 `profiles.access_level` 与 `courses.membership_type` 判断当前用户是否能访问课程；管理员直接通过。
- `public.course_questions_feed(course_id)`：前台读取问答时使用，返回隐私处理后的 `author_display_name`，避免放宽 `profiles` 表 RLS。
- 普通用户只能插入自己的可见问题/回复，不能更新或删除内容。
- 管理员可将问题/回复状态改为 `visible`、`hidden`、`deleted`。
- `deleted` 是软删除状态，前台不可见，数据保留。
