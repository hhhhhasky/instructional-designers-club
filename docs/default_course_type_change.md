# 课程页面默认显示调整 - 技术实现说明

## 更新时间
2025-11-13

## 需求概述

调整课程页面的默认显示逻辑，将初始加载时显示的课程类型从 **Pro 课程** 改为 **Plus 课程**。

## 技术实现

### 1. 核心修改

**文件**：`src/pages/CoursesPage.tsx`

**修改位置**：第 182 行

**修改内容**：
```typescript
// 修改前
const [selectedMembershipType, setSelectedMembershipType] = useState<MembershipType>('pro');

// 修改后
const [selectedMembershipType, setSelectedMembershipType] = useState<MembershipType>('plus');
```

### 2. 修改说明

#### 状态变量
- **变量名**：`selectedMembershipType`
- **类型**：`MembershipType`（'pro' | 'plus' | 'free'）
- **作用**：控制当前显示的课程类型
- **默认值**：从 `'pro'` 改为 `'plus'`

#### 影响范围
修改默认值后，以下功能会自动适配：

1. **数据加载**：`useEffect` 监听 `selectedMembershipType` 变化，自动加载对应的课程数据
2. **UI 显示**：会员类型按钮根据 `selectedMembershipType` 显示选中状态
3. **场景导航**：Plus 课程的场景导航区自动显示
4. **课程列表**：显示 Plus 会员的所有课程分类

## 功能流程

### 页面加载流程

```
用户访问课程页面
    ↓
CoursesPage 组件初始化
    ↓
selectedMembershipType 初始化为 'plus'
    ↓
useEffect 触发，加载 Plus 课程数据
    ↓
并行请求：
  - getCategoriesByMembershipType('plus')  // 获取分类列表
  - getCoursesByMembershipType('plus')     // 获取课程列表
    ↓
加载课程分类标签数据
  - getBatchCategoryTags(categories)       // 获取标签数据
    ↓
更新状态：
  - setCategories()          // 设置分类列表
  - setAllCourses()          // 设置课程列表
  - setCoursesByCategory()   // 设置分类分组
  - setCategoryTags()        // 设置标签数据
    ↓
渲染页面：
  - Plus 按钮显示为选中状态（青色渐变）
  - 场景导航区显示（4 个场景入口）
  - 课程分类列表显示
```

### 用户交互流程

```
用户点击 Pro 按钮
    ↓
setSelectedMembershipType('pro')
    ↓
useEffect 触发，加载 Pro 课程数据
    ↓
场景导航区自动隐藏（仅 Plus 显示）
    ↓
Pro 按钮显示为选中状态（橙色渐变）
    ↓
课程列表更新为 Pro 课程
```

## 数据流

### 状态管理

```typescript
// 会员类型状态
selectedMembershipType: 'plus'  // 默认值

// 场景筛选状态
selectedScenario: null  // 初始无筛选

// 课程数据状态
categories: string[]  // 分类列表
allCourses: Course[]  // 所有课程
coursesByCategory: Record<string, Course[]>  // 分类分组
categoryTags: Record<string, {...}>  // 标签数据

// UI 状态
isLoading: boolean  // 加载状态
error: string | null  // 错误信息
isNavigating: boolean  // 导航状态
clickedCourseId: string | null  // 点击的课程 ID
```

### API 调用

```typescript
// 1. 获取分类列表
getCategoriesByMembershipType('plus')
// 返回：['教学目标', '任务情境设计', '学情分析', ...]

//   },
//   ...
// }
```
